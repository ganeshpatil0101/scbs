# Reports Overview

## Purpose

Index of daily-use operational and statutory reports captured from `screenshots/Reports/`. Reports fall into two types:

| Type | Description | Examples |
| :--- | :--- | :--- |
| **Screen + Print** | Filter/search UI on screen; user runs **रिपोर्ट** to generate printable output | GL Wise, Purvani |
| **Print** | Printable layout only (generated from menu or from a parent screen) | Balance Sheet, Daybook, Trial Balance, P&L, Loan Statement |

Branch and GL lookups use a single **Autocomplete** field per entity — see [../shared/entity-autocomplete-pattern.md](../shared/entity-autocomplete-pattern.md).
